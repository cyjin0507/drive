--스프레드시트란--
표 계산은 표 형식으로 데이터의 조직, 분석, 저장을 가능케 하는 상호작용 컴퓨터 애플리케이션

--함수 정리--
HLOOKUP : 행을 참조할 때 사용 ex) =HLOOKUP(AVERAGE(G3:H3),$G$12:$K$14,k)
VLOOKUP : 열을 참조할 때 사용 ex) =VLOOKUP(AVERAGE(G3:H3),$G$12:$K$14,k)
SMALL : 묶은값 중에서 k번째로 작은 값 출력 ex) =SMALL(A1:A7,k)
LARGE : 묶은값 중에서 k번째로 큰 값 출력 ex) =LARGE(A1:A7,k)
ROUND : 선택한 값을 소수 K번째에서 반올림 ex) =ROUND(A5,k)
ROUNDUP : 선택한 값을 소수 K번째에서 올림 ex) =ROUNDUP(A5,k)
ROUNDDOWN : 선택한 값을 소수 K번째에서 내림 ex) =ROUNDDOWN(A5,k)
AVERAGE : 선택한 값들의 평균 ex) =AVERAGE(A1:A4)
LOWER : 선택한 문자열 모두 소문자로 변경 ex) =LOWER(G1)
PROPER  : 선택한 문자열 첫글자를 대문자로 변경 ex) =PROPER(G1)
CHOOSE : k번째에 있는 값을 출력 ex) =CHOOSE(2,"red","blue","green") -> blue
YEAR : 선택한 숫자(2021-01-01)에서 연도만 추출 ex) = =YEAR(G5)
TODAY : 오늘 날짜 출력 ex) =TODAY()
MOD : a / b의 나머지 ex) =MOD(a,b)
MID  : 선택한 문자열을 a번째부터 b만큼 추출 =MID("korea",1,2) -> ko
IF : 조건 ex) =IF(A1=12,"1","2") -> A1이 12이면 1 아니면 2
OR : 또는 조건 ex) =OR(A1=12,A2=24)
AND : 그리고 조건 ex) =AND(A1=12,A2=24)


--스프레드시트 정리--
-통합 : 형식이 같은 표를 합친다
-가상분석 : 셀 값을 변경하여 이러한 변경 내용이 워크시트의 수식 결과에 어떤 영향을 미치는지를 확인 (시나리오, 목표값 찾기)
-조건부서식 : 셀 안의 내용이 어떤 조건에 해당 하면 그 셀 또는 셀이 속해있는 행 전체에 대해서 서식(스타일)을 자동으로 지정
-텍스트 나누기 : 셀 안의 내용을 어떤 기준을 두고 텍스트로 나누는것
-데이터 유효성 검사 : 셀에 값을 입력할 때 사용자가 원하는 값으로 입력되었는지 체크를 알아서 해주는 기능
-목표값 찾기 : 특정한 결과값을 얻으려면 어떤 입력값을 입력해야 하는지 알 수 있는 기능
-부분합 : 데이터를 특정 필드를 기준으로 그룹화하여 각 그룹에 대한 합계, 평균, 개수, 최대값, 최소값, 표준편차, 분산 등의 통계를 자동으로 계산해 요약하는 기능
